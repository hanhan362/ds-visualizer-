# DS-Visualizer 需求文档

## 1. 项目目标

开发一个数据结构可视化教学平台，通过逐步动画直观展示数据结构和算法的执行过程，帮助学生理解抽象概念。

## 2. 功能需求

### 2.1 可视化模块（8 个）

| 模块 | 路由 | 操作 | 渲染引擎 |
| --- | --- | --- | --- |
| 数组 | `/array` | push / pop / shift / unshift | SVG 柱状图 |
| 链表 | `/linked-list` | 头插 / 尾插 / 头删 / 尾删 / 反转 | SVG 节点 + 箭头 |
| 栈/队列 | `/stack-queue` | 栈（push/pop）队列（enqueue/dequeue） | SVG 柱状图 |
| 排序算法 | `/sort` | 冒泡 / 选择 / 插入 / 快速 / 归并 | Canvas 柱状图 |
| 二叉树 | `/tree` | 前序 / 中序 / 后序遍历 | SVG 树 |
| 图算法 | `/graph` | BFS 广度优先 / DFS 深度优先 | SVG 节点连线 |
| 学习中心 | `/learning` | 学习进度追踪 | 进度条 |
| 个人中心 | `/profile` | 操作历史 / 收藏 / 笔记 / 进度 | 四 Tab |

## 3. 非功能需求

### 3.1 性能

| 指标 | 目标 |
| --- | --- |
| 前端构建 | < 30s |
| 页面加载 | < 3s（含 Element Plus） |
| API 响应 | < 100ms |
| Canvas 帧率 | 60fps |

### 3.2 安全

- 密码 BCrypt 加密存储
- JWT Token 24h 过期
- CORS 白名单
- Prisma 参数化查询防注入
- Vue 默认 XSS 防护

### 3.3 兼容性

- Chrome / Firefox / Edge 最新版
- GitHub Pages 静态托管
- 响应式布局

### 3.4 可维护性

- TypeScript 类型完整
- 组件化开发（Vue SFC）
- 统一 API 响应格式 `{code, message, data}`
- 代码注释（关键逻辑）

## 4. 技术选型理由

| 技术 | 理由 |
| --- | --- |
| Vue 3 | 响应式数据绑定，适合高频动画更新 |
| TypeScript | 类型安全，数据结构场景下必不可少 |
| Pinia | 比 Vuex 更轻量的状态管理 |
| Element Plus | 成熟的中后台 UI 组件库 |
| Tailwind CSS | 原子化 CSS，快速布局 |
| Canvas | 高频重绘场景（排序动画）性能最优 |
| SVG | 交互式结构（节点/边）天然支持事件 |
| Express | 与前端同语言，代码复用 |
| Prisma | 类型安全 ORM，自动映射现有表 |
| MySQL | 持久化数据存储 |
| JWT | 无状态认证，适合前后端分离 |
